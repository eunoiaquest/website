# **App Name**: Eunoia Quest Landing Page

## Core Features:

- Navigation Bar: Implement a sticky header/navigation bar with links to different sections of the page (Home, About, Features, Team, Contact).
- Section Layout and Content: Design and implement the Hero, About, Features, Team, and Contact sections as described.
- Page Animations: Add smooth scrolling, fade-in, and slide-up animations on scroll for enhanced user experience.

## Style Guidelines:

- Background: White
- Text: Charcoal-gray
- Accent: #65558f on buttons, links, and icons
- Secondary accent color: Light shade of purple (#A9A1CE) for subtle highlights.
- Headings: Quicksand Medium
- Body: Quicksand Light or Quicksand Ultralight
- Use Flexbox and Grid for modern, responsive layouts.
- Ensure generous padding and margins for a clear visual hierarchy.
- Implement smooth scrolling between sections.
- Add subtle fade-in and slide-up effects on scroll for each section.
- Line-art style icons for features section.
